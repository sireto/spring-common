# Spring Common

This project is a collection of common classes useful in Spring Boot development.

### Includes
* JWT Decoder, Interceptor, and SessionService
* Entities base classes
* List of common base exception classes
* Pagination Spec
* Utility classes
  * DateUtils
  * FileUtils
  * GeneralUtils
  * MathUtils
  * MockUtils
  * RandomGenUtils
  * RequestUtils
  * ResponseParserUtils
  * StringUtils
